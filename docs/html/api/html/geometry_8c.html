EmbRect embGeometry_boundingRect ( EmbGeometry * obj)

Calculate the bounding box of geometry obj based on what kind of geometric object it is.

obj A pointer to the geometry memory.

Returns
EmbRect The bounding box in the same scale as the input geometry.

In the case of a failure the bounding box returned is always the unit square with top left corner at (0, 0).
