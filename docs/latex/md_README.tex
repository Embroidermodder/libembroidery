Libembroidery is a low-\/level library for reading, writing, and altering digital embroidery files in C. It is part of the Embroidermodder Project for open source machine embroidery.

Libembroidery is the underlying library that is used by \href{http://embroidermodder.org}{\texttt{ Embroidermodder 2}} and is developed by The Embroidermodder Team. A full list of contributors to the project is maintained in \href{https://github.com/Embroidermodder/embroidermodder}{\texttt{ the Embroidermodder 2 github}} in the file {\ttfamily CREDITS.\+md}. It handles over 45 different embroidery specific formats as well as several non-\/embroidery specific vector formats.

It also includes a CLI called {\ttfamily embroider} that allows for better automation of changes to embroidery files and will be more up-\/to date than the Embroidermodder 2 GUI.\hypertarget{md_README_autotoc_md35}{}\doxysection{Documentation}\label{md_README_autotoc_md35}
For basic usage, {\ttfamily embroider} should have some in-\/built help starting with\+:
